# Vitrine pour le business

## ⭐️ Qu'est-ce que c'est ??

[Vitrine pour business](https://land-bit.github.io/Vitrine_business/) - Exemplaire d'un site web pour demo de votre entreprise.

**NOTE: Ce site à était fait pour un but educatif!**

## ✨ Rendu

Vous trouverez ci-dessous une liste des principales vues prévues :

- Création d'une page HTML, avec les balises et les atributs -- **(Pour la comprehension de balise HTML et les atribut)**
- Les 3 utilisations principales du CSS dans un site web -- **(Pour la comprehension du CSS et son foctionnement)**
- Utilisation des selecteurs CSS -- **(Pour la comprehension et utilisation des selecteurs CSS)** 
- Utilisation des pseudo-élements CSS -- **(Pour la comprehension et utilisation des pseudo-elements CSS)**
- Utilisation de Git pour la gestion du projet -- **(Pour la comprehension du gestion du projet)**
- Utilisation de GitHub pages pour faire la demo de mise en production d'un site web -- **(Pour la demo d'un site en production)**

## 📄 Contenus

-   `/images/` - Un repertoire où se trouve tous les images et les icons du site
-   `/index.html` - Un fichier qui contient le html de notre page
-   `/style.css` - Un fichier qui contient du css
